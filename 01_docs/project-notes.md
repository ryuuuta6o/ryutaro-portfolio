## デザイン

- コンセプト: FIELD SIGNAL CORE
- 方向性: ダーク、工業的、発光シアン、情報可視化
- 3Dコア: 現場情報の収集、接続、再構成を視覚化
- 情報設計: 実績は「課題・実装・成果」、プロフィールは「観察・整理・実装」の順で理解できる構造
- 写真: トリミングせず各画像の原寸比率を維持
- モバイル: 3Dの粒子数と解像度を抑え、実績は縦スクロールで全件を閲覧できる一覧に変更
